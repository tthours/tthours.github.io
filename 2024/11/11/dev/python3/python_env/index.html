Anaconda

简介

Anaconda是一个开源的Python包管理器，而Miniconda则是轻量级的Anaconda

下载地址

最新版本：https://docs.conda.io/en/latest/miniconda.html

历史版本：https://repo.anaconda.com/miniconda/

安装

wget https://repo.anaconda.com/miniconda/Miniconda3-latest-Linux-x86_64.sh

chmod +x Miniconda3-latest-Linux-x86_64.sh

./Miniconda3-latest-Linux-x86_64.sh

重新加载环境变量文件或重启电脑

测试是否安装成功

conda –version

命令

创建环境

conda create -n condaName python=版本

例如：

conda create -n open_manus python=3.12

激活环境

conda activate condaName

例如：

conda activate open_manus

退出环境

conda deactivate

查看当前环境中存在的虚拟环境

conda env list

或

conda info -e
