top

前言

默认自带该命令

用于实时动态的查看系统的整体运行情况

命令

语法

top(选项)

选项

-b : 以批处理模式操作

-c : 显示完整的命令

-d : 屏幕刷新间隔时间 -d ss.tt (seconds.tenths)

-I ：忽略失效过程

-s ：累积模式

-i<时间> : 设置间隔时间

-u<用户名>：指定用户名

-p<进程号>：指定进程

-n<次数>：循环显示的次数

-o [+/-]fieldname1 ：指定排序字段，+表示降序，-表示升序

VIRT

virtual memory usage 虚拟内存

进程需要的虚拟内存大小

如果进程申请 100M 内存，实际只使用 10 M ，值会增长 100M

RES

resident memory usage 常驻内存

进程当前使用的内存大小，不包括 swap out

包含其他进程的共享

只统计加载的库文件所占内存大小

如果进程申请 100M 内存，实际只使用 10M ，值只会增长 10M，与 VIRT 相反

SHR

shared memory 共享内存

除了自身进程的共享内存，也包括其他进程的共享内存

虽然进程只使用几个共享库的函数，但是包含了整个共享库的大小

某个进程所占物理内存大小： RES - SHR

swap out 后，值会下降

这个值是 CPU 内存 硬盘 综合的负载
