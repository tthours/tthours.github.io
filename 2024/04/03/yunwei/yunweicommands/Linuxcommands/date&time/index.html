data&time

简介

获取和设置主机日期时间

常用操作

获取主机当前时间

date

date -R

修改日期

需要root权限

sudo date -s “YYYY-MM-DD hh:mm[:ss]”

修改时间

需要root权限

sudo date -s hh:mm[:ss]

更新时间

安装 ntpdate

更新时间

sudo ntpdate 时间服务器域名/ip

报错信息

定时任务周期性更新时间

时间自动更新配置

查看时间自动更新配置

timedatectl show

NTP参数标志是否自动更新

yes ：自动更新

no：不自动更新

开启自动更新

timedatectl set-ntp yes
