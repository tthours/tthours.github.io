ifdown

禁用网卡

查询网卡信息

使用ifconfig，不要使用ip a

ip a 命令查询出来的网卡名存在问题

同台机器

ip a : eth0.200@eth0

ifconfig : eth0.200

示例：

ifconfig

ip a

禁用网卡

语法：

ifdown

ifconfig down

启用网卡

语法：

ifup

备注

网络不通

现象

操作网卡可能会导致对应网卡路由丢失，从而导致网络不通

解决方法

备份路由，恢复路由

1、操作网卡之前备份路由，恢复路由

route 或 route -n

2、如果没有备份路由，可以尝试查看/etc/rc.local文件找路由信息

执行对应的路由操作即可
