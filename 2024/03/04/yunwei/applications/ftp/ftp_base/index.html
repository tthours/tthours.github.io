ftp_base

简介

FTP（File Transfer Protocol）文件传输协议，用于在网络上进行文件传输的协议

安装

Linux

Linux下常用的ftp工具为vsftp

检查是否安装

rpm -qa | grep vsftpd

yum安装

yum install vsftpd -y

操作

启动

service vsftpd start

systemctl start vsftpd

停止

service vsftpd stop

systemctl stop vsftpd

状态

service vsftpd status

systemctl status vsftpd

默认信息

默认可执行文件

/usr/sbin/vsftpd

默认安装配置目录

/etc/vsftpd/

默认核心配置文件

/etc/vsftpd/vsftpd.conf

配置说明

anonymous_enable=YES # 允许匿名登录

local_enable=YES # 允许本地用户登录

write_enable=YES # 允许写操作

local_umask=022 # 新增文件权限，默认022

默认日志

yum 安装默认没有开启日志功能，需要额外配置

常见日志路径/var/log/vsftpd.log

日志开启配置

ftp远程连接

Linux

ftp命令

命令安装

yum install ftp -y

连接

ftp ftp.example.com

如果支持匿名访问，用户名可以输入anonymous，密码为邮件地址或者空

lftp命令

命令安装

连接默认端口

lftp ftp://ftp.example.com

连接指定端口

lftp ftp://ftp.example.com:port

ftp操作命令

ls/dir

列出当前路径下文件及文件夹

cd dirname

切换目录

put local-file remote-fil

quit/bye

退出会话

lftp 1.1.1.1:/pub> quit (base) [root@nvm /]# lftp ftp://1.1.1.1 lftp 1.1.1.1:~> bye (base) [root@nvm /]#
