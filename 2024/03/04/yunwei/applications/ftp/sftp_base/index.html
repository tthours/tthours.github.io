sftp_base

SSH文件传输协议/安全文件传输协议

SSH File Transfer Protocol

Secret File Transfer Protocol

简介

使用ssh协议进行文件传输

默认端口

22

安装

Linux

sftp是基于ssh协议，安装OpenSSH软件即可

Ubuntu、Debian服务器

dpkg -l | grep ssh

ii 表示已安装的软件包

redhat、Centos等服务器

rpm -qa | grep ssh

使用

登录

sftp [user@]host[:dir[/]]

sftp [user@]host[:dir[/]] -p port

上传

基于登录情况下，也就是sftp终端界面

上传本地文件到远程目录

put localfile [remotedir]

如果不指定目录，上传到当前所在目录

如果指定目录，可能会存在当前登录账号权限不足的问题

上传本地目录文件到远程目录

put -r localDir remoteDir

上传多个文件到远程目录

文件名支持通配符*

mput localfile remoteDir

上传多个目录的文件到远程目录

目录名支持通配符*

mput -r locatDirs remoteDir

下载

下载远程文件到本地目录

get remoteFile [localDir]

下载远程目录文件到本地目录

get -r remoteDirFiles [localDir]

下载多个远程文件到本地目录

文件名支持通配符*

mget remoteFiles [localDir]

下载多个远程目录的文件到本地目录

目录名支持通配符*

mget remoteDirsFile [localDir]

执行远程命令

删除 rm

没有 -r -f 这种参数，每次也只能删除一个文件，多个文件名只删除第一个

列表 ls

权限修改 chmod

执行本地命令

sftp命令-常规命令前加 l

lls
lpwd

本地命令-本地存在命令前加 !

!dir

!ipconfig
