安装基础环境

安装npm

brew install npm

npm -v 查看已安装版本

安装node

brew install node

node -v 查看已安装版本

安装hexo-cli

npm install -g hexo-cli

博客部署

初始化

hexo init 博客文件夹

初始化之后会在当前目录创建 博客文件夹，文件夹中会存在以下目录及文件

本地测试

进入创建的 博客文件夹

cd tthourshexo

创建文件

hexo new hexo_install

静态化处理

hexo g

启动服务预览

hexo s

本地测试

http://localhost:4000/
