2022.9.28

有人想不通贾跃亭为什么到今天还要造车。

贾跃亭必须造车。普京必须打乌克兰。乾隆必须凑十全武功。这个用咱们过日子人的心去想是想不明白的。

好大喜功和雄才伟略之间的区别主要是干没干成。没干成就是好大喜功，干成了就是雄才伟略。

所以前面的例子只能举到乾隆。因为十全武功虽然凑成了，但比较勉强。后世夸得少，笑的多。所以说乾隆好大喜功不会有很多人反对。要是举了秦皇汉武的例子，挨骂的就是我了。我们从来不用“好大喜功”形容成功者。

其实，如果一个人不“好大喜功”，或者用现代词儿说叫“强烈的成就动机”，又怎么能做成大事呢？靠狗屎运吗？所以，单单“好大喜功”其实没什么不对。成功人士都好大喜功。

当然，说“好大喜功”等于“雄才伟略”其实也不对。“好大喜功”只对应了“伟略”。有伟略，还得有雄才。那么，即便最后没成，大家也认为是时也运也命也，不至于给个好大喜功的评价。诸葛亮五次北伐都没搞成，后人也不说他好大喜功，只说他鞠躬尽瘁。

根据国家疾控中心最新一期流感监测周报，北方的流感样病例比南方严重很多。不过总体来说都比去年要轻。去年这个时候不仅比今年更严重，而且南方北方都很严重。

今年南北方流行的流感毒株基本都是 A(H1N1)pdm09。今年的三价、四价流感疫苗都覆盖了这个毒株。所以如果之前打了疫苗，应该能从中受益。

另外，除了极少数 A(H1N1)pdm09 毒株对神经氨酸酶抑制剂敏感性高度降低外，其它毒株对神经氨酸酶抑制剂和聚合酶抑制剂都敏感。简单说就是奥司他韦、玛巴洛沙韦都还好使。

所有 Windows 版本都通用的就是 Print Screen 截全屏以及 Alt + Print Screen 截当前窗口。Windows 10/11 支持更高级的截屏方式，可以自选截屏区域。按 Win + Shift + S 即可启动该功能。具体可参考：https://support.microsoft.com/zh-cn/windows/%E4%BD%BF%E7%94%A8%E6%88%AA%E5%9B%BE%E5%B7%A5%E5%85%B7%E6%8D%95%E8%8E%B7%E6%88%AA%E5%9B%BE-00246869-1843-655f-f220-97299b865f6b

不过以上这些截屏方式都是把截取的内容存在剪贴板。如果希望直接把截屏存为文件，可以用 Windows 自带的 Game Bar 截屏，Win + Alt + Print Screen，截屏会自动存储到用户目录的 Videos\Captures。

另外，Game Bar 还可以录制屏幕录像，Win + Alt + R 即可启动，同样保存在用户目录的 Videos\Captures。也就是说，在 Windows 10/11 上录制屏幕录像并不需要依赖第三方软件。
