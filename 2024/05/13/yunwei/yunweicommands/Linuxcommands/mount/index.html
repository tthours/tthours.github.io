mount

mount -t nfs -o vers=3,timeo=600,nolock,rsize=1048576,wsize=1048576,hard,retrans=3,noresvport,async,noatime,nodiratime scan.security.nfs.cu-beijing-1-a.nas.aa.bb.com.cn:/share ./sfs

-t fstype

指定文件系统类型

支持的文件系统类型可以查看 /proc/filesystems或 /lib/modules/$(uname -r)/kernel/fs

cat /proc/filesystems

ls /lib/modules/$(uname -r)/kernel/fs

-o，–options opts

指定特殊的选项

async

I/O 异步传输

atime

不使用noatime特性，因此inode访问时间由内核默认值控制

noatime

不更新文件访问时间（access time）

nodiratime

不更新目录访问时间（access time）

diratime

更新目录访问时间，默认配置，如果设置noatime，该配置忽略

nolock
noresvport
vers
timeo
rsize

设置读缓冲区字节大小

rsize=1048576

wsize

设置写缓冲区字节大小

wsize=1048576

hard
retrans

error

nfs缺少帮助程序

报错信息

mount -t nfs -o async,noatime,nodiratime scan.nas.baidu.com:/share ./sfs

排查过程

根据错误提示可知，缺乏/sbin/mount.nfs程序

在其他服务器上查看存在该程序

解决方案

sudo yum install nfs-utils

sudo dnf install nfs-utils

yunwei > commands
