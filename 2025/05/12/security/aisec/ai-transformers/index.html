transformers

https://github.com/huggingface/transformers/blob/main/i18n/README_zh-hans.md

Hugging Face 出品的 工具包，实现了大量的主流预训练

使用 PyTorch 或 TensorFlow 快速训练一个模型

原理

安装

创建虚拟环境

conda create -n transformers python=3.12

conda activate transformers

pip 安装

pip install transformers

报错

缺乏PyTorch、TensorFlow、Flax

使用

classifier = pipeline(‘sentiment-analysis’) 下载预训练 sentiment-analysis 模型

pipeline

推理 API

支持 GPU、Apple Silicon 和半精度权重

语法

pipeline = pipeline(“text-generation”, model=model_name, device=”cuda”)

model：指定使用的模型名称或模型对象

tokenizer：指定使用的分词器名称或分词器对象

device：指定运行设备，如 'cpu' 或 'cuda'，GPU推理时，device=0 or device=1 or device=2(数字为选择第几张显卡);当使用CPU推理时，device=cpu

batch_size：批处理大小，适用于批量处理输入

支持的任务列表

任务 描述 模态 Pipeline
文本分类 为给定的文本序列分配一个标签 NLP pipeline(task=“sentiment-analysis”)
文本生成 根据给定的提示生成文本 NLP pipeline(task=“text-generation”)
命名实体识别 为序列里的每个 token 分配一个标签（人, 组织, 地址等等） NLP pipeline(task=“ner”)
问答系统 通过给定的上下文和问题, 在文本中提取答案 NLP pipeline(task=“question-answering”)
掩盖填充 预测出正确的在序列中被掩盖的token NLP pipeline(task=“fill-mask”)
文本摘要 为文本序列或文档生成总结 NLP pipeline(task=“summarization”)
文本翻译 将文本从一种语言翻译为另一种语言 NLP pipeline(task=“translation”)
图像分类 为图像分配一个标签 Computer vision pipeline(task=“image-classification”)
图像分割 为图像中每个独立的像素分配标签（支持语义、全景和实例分割） Computer vision pipeline(task=“image-segmentation”)
目标检测 预测图像中目标对象的边界框和类别 Computer vision pipeline(task=“object-detection”)
音频分类 给音频文件分配一个标签 Audio pipeline(task=“audio-classification”)
自动语音识别 将音频文件中的语音提取为文本 Audio pipeline(task=“automatic-speech-recognition”)
视觉问答 给定一个图像和一个问题，正确地回答有关图像的问题 Multimodal pipeline(task=“vqa”)

AutoClass

AutoModelForSequenceClassification 和 AutoTokenizer 一起支持在上面用到的 pipeline()

AutoClass 是一个能够通过预训练模型的名称或路径自动查找其架构的快捷方式

AutoTokenizer

分词器负责预处理文本，将文本转换为用于输入模型的数字数组

tokenizer
语法

max_length：模型支持的最大序列长度，超过后会被截断

padding： True

自动对批次中的文本进行填充（Padding），使所有序列长度一致

填充规则：

  • 按照 padding_side ，默认为 right，在右侧填充
  • 填充符号由 pad_token 决定

truncation：True

自动截断超过 max_length 的序列

截断规则：

  • 按照 truncation_side ，默认为 right ，从右侧截断
  • 截断后保留的 token 数量不超过 max_length

return_tensors

指定返回张量的框架类

可选值：

  • pt：返回 PyTorch 张量（torch.Tensor）

    返回值中包含 input_ids、attention_mask 字段

    input_ids：每个 token 的 ID（包括特殊 token 如 [CLS] 和 [SEP]）

    attention_mask：标记哪些位置是有效 token（1），哪些是填充（0）

  • tf：返回 TensorFlow 张量（tf.Tensor）

  • np：返回 NumPy 数组（np.ndarray）
