rpm

说明

在基于 RPM（Red Hat Package Manager）的 Linux 发行版（如 Red Hat Enterprise Linux、CentOS、Fedora 等）中，你可以使用 rpm 命令来查询已安装的软件包

语法命令

列出所有已安装的软件包

-q 查询模式

-a 列出所有已安装的软件包

模糊匹配安装的软件包

grep -i 正则匹配

查询某个软件包详细信息

Name：软件包名称

Version：软件包版本

Release：发行版本

Install Date：安装时间

Group：所属类别

Size：空间占用大小

License：许可证类型，Commercial-商业的

Signature：签名信息

Source RPM：RPM 包名称

列出指定 RPM 包安装的所有文件

查询某个文件属于的 RPM 包

验证已安装的包

验证已安装包的文件是否有任何修改、删除或权限变更

如果有输出，则表示存在改动，常见结果说明

  • S：size 文件大小不同
  • 5：md5 MD5 校验和不匹配
  • T：time 修改时间不同
  • G：group 组不同
  • U：user 所有者不同
  • L：link 链接路径不同
  • M：文件权限或类型不同
  • D：设备号不同
