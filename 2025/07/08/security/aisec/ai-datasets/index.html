ai-datasets

描述

datasets 模块通常指的是 Hugging Face 提供的 datasets 库（全称：Datasets），这是一个用于轻松访问和共享自然语言处理（NLP）以及其他领域数据集的工具库。它由 Hugging Face 社区开发和维护，与 Hugging Face 的 transformers 库紧密集成，广泛用于机器学习和自然语言处理任务。

官方文档

https://huggingface.co/docs/datasets

函数

load_dataset() 加载数据集

语法

dataset_name：标识需要加载的数据集名称

name：指定数据集不同的配置

​ plain_text - 纯文本格式，默认格式

​ bytes - 原始字节格式

​ tokens - 已分词格式，很少使用

split：用于指定需要加载的数据集划分

​ test - 测试集

​ train - 训练集

返回值

dataset 是 Dataset 对象

类似于一个表格（类似 Pandas DataFrame），每一行是一条样本（sample），每一列是一个特征（feature）

每条 sample 包含两列，test 和 label

test 格式为 str 类型，是 电影评论的文本内容（字符串）

label 格式为 int 类型，是 电影评论的情感标签，0 表示负面（negative），1 表示正面（positive）
